What does Code Clone / Duplicate Code mean?

Duplicate code is a computer programming term for a sequence of source code that occurs more than once, either within a program or across different programs owned or maintained by the same entity.

source : Wikipedia

Type of Clones

In this paper they classify as four clone types.

Type I clones: Identicalfragments except for variations in comments and layout

Type II clones: Identical fragments except for variations in iden-tifier names and literal values in addition to Type I differences

Type III clones: Syntactically similar fragments that differ at the statement level

Type IV clones: Syntactically dissimilar fragments that implement the same functionality

Why do we need Softwares for Code Clone Detection?
There are many reasons why we need to detect code clones

Some of them are listed below:

  1. To detect plagariasm in codebase
  2. There can be bugs caused by two code clones
  3. Searching for Refactoring code when the codebase is too big

Deep Learning Based Code Clone Detection

The approach uses Deep Learning namely two Recurrent Neural Networks to detect clones.

This approach is different from the other approaches because it does not depend on handcrafted features to represent code.

The other approaches depends on features like :

  • Frequency of Words
  • Indentation pattern
  • Length of fragments in source code
